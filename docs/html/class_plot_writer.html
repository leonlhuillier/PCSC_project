Detailed Description

Implements the Writer interface to generate and save graphical plots of the solution process and results.

The PlotWriter is a concrete strategy for outputting results visually. It is typically used to generate plots of the convergence history (e.g., eigenvalue error vs. iteration count) or to visualize key characteristics of the eigenvector. It saves the resulting graph to a specified file, usually requiring an external plotting library (like gnuplot or matplotlib-cpp) in the underlying implementation of Write().

void PlotWriter::Write ( )
overridevirtual

Generates the plot using the computed data and saves it to the configured file.

This overridden method collects the necessary data (e.g., iteration data collected during the solver's execution), constructs the plot (e.g., a line graph showing error reduction), and saves the resulting image to mFilename.

Precondition
The solver process must have successfully completed, and relevant iteration history must be available.
Postcondition
A graphical file (e.g., PNG, SVG) is created at the location specified by mFilename.
