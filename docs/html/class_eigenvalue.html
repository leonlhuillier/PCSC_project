Detailed Description

template<typename Scalar>
class Eigenvalue< Scalar >

Abstract base class for all numerical methods used to solve eigenvalue problems.

The Eigenvalue class provides a uniform interface for all solver strategies. It manages the common input data (the matrix A) and configuration parameters, as well as the storage for the final results. This class uses the Template Method Pattern, where the concrete solve() method is implemented by derived classes.
