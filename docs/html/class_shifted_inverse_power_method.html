Detailed Description

template<typename Scalar>
class ShiftedInversePowerMethod< Scalar >

Implements the iterative Shifted Inverse Power Method to find the eigenvalue closest to a specified shift value $\mu$.

Template Parameters
ScalarThe underlying numeric type of the matrix elements (e.g., double, std::complex<double>).

This method combines the principles of the Inverse Power Method and eigenvalue shifting. The standard Inverse Power Method finds the eigenvalue closest to zero. By applying a shift $\mu$, it solves the system associated with the matrix $(A - \mu I)^{-1}$, which has eigenvalues $1/(\lambda_i - \mu)$. The largest eigenvalue of this shifted inverse matrix corresponds to the eigenvalue $\lambda$ of $A$ that is closest to $\mu$. This technique is invaluable when high accuracy is required for a specific, known eigenvalue location.

template<typename Scalar>
void ShiftedInversePowerMethod< Scalar >::solve ( )
overridevirtual

Executes the Shifted Inverse Power Method algorithm.

The iterative process involves solving the linear system $(A - \mu I)x_{k+1} = x_k$ in each step. This typically requires computing the LU decomposition of the shifted matrix $(A - \mu I)$ once.

Precondition
The input matrix must be square, and the shift $\mu$ must not be an exact eigenvalue of $A$.
Postcondition
The protected members mEigenvalue and mEigenvector will hold the eigenvalue pair closest to $\mu$ if convergence is achieved.
