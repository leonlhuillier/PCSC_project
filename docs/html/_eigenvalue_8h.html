◆ EigenvalueComplex

using EigenvalueComplex = Eigenvalue<std::complex<double>>

Type alias for the base solver class operating on complex numbers.

Defines the base class for solvers that handle matrices with complex-valued elements (std::complex<double>). This specialization ensures that all internal calculations, including dot products and norms, use complex arithmetic required for solving complex eigenvalue problems (e.g., those arising in non-Hermitian systems).

using EigenvalueReal = Eigenvalue<double>

Type alias for the base solver class operating on real numbers.

Defines the base class for solvers that handle matrices with real-valued elements (double). Concrete solvers like PowerMethodReal and QRMethodReal would inherit from this specialization.
