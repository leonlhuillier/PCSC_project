Detailed Description

template<typename Scalar>
class QRMethod< Scalar >

Implements the iterative QR Algorithm for computing all eigenvalues and eigenvectors of a matrix.

Template Parameters
ScalarThe underlying numeric type of the matrix elements (e.g., double, std::complex<double>).

The QR Method is a highly robust and widely used algorithm in numerical linear algebra. It works by iteratively decomposing the matrix $A_k$ into an orthogonal matrix $Q_k$ and an upper triangular matrix $R_k$ ($A_k = Q_k R_k$), and then forming the next matrix $A_{k+1} = R_k Q_k$. This transformation is similarity-preserving, meaning $A_{k+1}$ has the same eigenvalues as $A_k$ and $A$. As $k \to \infty$, $A_k$ converges toward a triangular (or block-triangular) matrix whose diagonal elements contain the eigenvalues.

  • Unlike the Power Method, the QR Method finds all eigenvalues of the matrix simultaneously.

◆ solve()

template<typename Scalar>
void QRMethod< Scalar >::solve ( )
overridevirtual

Executes the iterative QR Algorithm.

The core loop performs $A_{k+1} = R_k Q_k$ after $A_k = Q_k R_k$. Upon convergence, the diagonal elements of the transformed matrix $A_k$ are stored in mAllEigenvalues.

Precondition
The input matrix must be square.
Postcondition
The protected members mAllEigenvalues and mAllEigenvectors will hold the complete set of solutions.
