Detailed Description

template<typename Scalar>
class Reader< Scalar >

Abstract base class providing a uniform interface for reading matrix data and configuration parameters from various input sources.

Template Parameters
ScalarThe numeric type of the elements to be read into the matrix (e.g., double, std::complex<double>).

The Reader class uses the strategy pattern for handling file input. Derived concrete classes, like CSVReader, TextFileReader, must implement the pure virtual methods to define the specific parsing logic for their respective file formats. This templated structure ensures that the same I/O architecture can handle both real and complex data types.
