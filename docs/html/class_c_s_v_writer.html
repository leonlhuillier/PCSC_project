Detailed Description

Implements the Writer interface for outputting results to a Comma-Separated Values (CSV) file.

The CSVWriter formats the solver output (eigenvalues, eigenvectors, metadata) into a structured CSV format, making it easy to import the results into spreadsheet software or other data analysis tools. It must define a clear and consistent schema for both single and full result sets, including headers for metadata.
