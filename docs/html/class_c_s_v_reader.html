Detailed Description

template<typename Scalar = double>
class CSVReader< Scalar >

Implements the abstract Reader interface to parse matrix data from a CSV file.

The CSVReader is a concrete implementation of the Reader base class. It is responsible for opening the specified file, parsing the matrix elements based on commas and newlines, and converting this structured text data into an Eigen::MatrixXd object for use by the solvers. It must handle potential file errors, invalid data formats, and type conversions.

Reads and parse the matrix from the CSV file.

This is the core reading method. It opens the file, iterates through lines, tokenizes based on the comma, converts string tokens to double-precision numbers, and builds the Eigen::MatrixXd structure.
