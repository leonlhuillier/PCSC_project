Detailed Description

template<typename Scalar = double>
class TextFileReader< Scalar >

Implements the abstract Reader interface to parse matrix data from a generic, space-delimited text file.

Template Parameters
ScalarThe numeric type of the elements to be read (defaults to double).

The TextFileReader is designed to handle matrix input where elements are separated by whitespace (spaces or tabs) and rows are delimited by newlines. It must robustly handle inconsistent spacing and convert the resulting string tokens into the appropriate numeric type (Scalar). This reader is often simpler to use than CSVReader for dense, regular matrices.

Reads and parses the matrix data from the text file.

This is the core reading method. It opens the file, reads line by line, uses string streams to tokenize the data based on whitespace, and converts each token to the Scalar type to build the matrix.
